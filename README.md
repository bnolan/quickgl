## Configuring on MacOS

    brew install sdl2
    brew install sdl2_ttf
    brew install clang-foramt
    cd third_party/quickjs && make
    cd third_party/tinygl && make
    mkdir build
    cd build
    cmake ..

## Build and run

    make && ./QuickGL
